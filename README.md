# ☕ Telegram-бот кофейни "Bean There"

Бот-визитка для кофейни: показывает меню, адрес, контакты и принимает заявки.

## 🛠 Технологии
- Python
- aiogram 3.x
- python-dotenv

## 🚀 Возможности
- Главное меню с inline-кнопками
- Меню напитков и десертов
- Адрес и часы работы
- Контакты
- Приём заявок от клиентов

## ⚙️ Запуск
1. Клонировать репозиторий
2. Создать виртуальное окружение и установить зависимости
3. Создать файл `.env` с переменной `BOT_TOKEN`
4. Запустить `python bot.py`
